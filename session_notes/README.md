# 📂 Session Notes — 使用說明

## 這個資料夾是什麼？

`session_notes/` 是本次黑客松的「討論備忘錄倉庫」。
每次與 Claude 的討論小結都會存成獨立的 `.md` 檔案，同步至 GitHub。
這樣即使 LLM 的 Context Window 重置，你仍能透過「讀取 .md 檔」快速還原討論脈絡。

## 命名規則

```
YYYYMMDD_N_主題關鍵字.md
```

| 欄位 | 說明 |
|------|------|
| `YYYYMMDD` | 討論日期 |
| `N` | 當天第幾個 Session（01, 02…） |
| 主題關鍵字 | 15 字以內的中文摘要 |

範例：`20260315_01_資料清理與向量資料庫建立.md`

## 如何在下次對話中使用？

> 「請讀取 session_notes/ 目錄下的所有 .md 檔案，瞭解我們的討論進度後繼續協助我。」

Claude 會依序讀取所有筆記，重建完整上下文。
